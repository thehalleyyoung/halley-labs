# Review: CABER — Coalgebraic Behavioral Auditing of Foundation Models

**Reviewer:** Lila Zhang
**Persona:** Symbolic Reasoning & AI Expert
**Expertise:** Neuro-symbolic AI, symbolic reasoning, knowledge representation, ontological reasoning, compositional reasoning, knowledge graphs

---

## Summary

CABER represents one of the most sophisticated attempts to apply algebraic and logical reasoning to LLM behavioral analysis. The coalgebraic framework provides genuine structural advantages over purely statistical approaches—the learned automata are interpretable symbolic models with ≤40 states and ≥92% prediction accuracy, the QCTL_F temporal specifications are compositional, and the Kantorovich bisimulation distances have rigorous algebraic foundations grounded in optimal transport. However, the symbolic reasoning remains fundamentally shallow: no abductive explanation of why behavioral divergence occurs, no compositional semantic analysis of prompts, no integration with the LLM's own symbolic processing capabilities, and the keyword-based classifier that grounds the symbolic abstraction is a weak link that limits the depth of all downstream symbolic reasoning.

## Strengths

1. **Symbolic behavioral models with genuine interpretability.** The learned behavioral automata provide interpretable, finite-state symbolic descriptions of LLM behavior that are qualitatively richer than statistical summaries. The discovery of an emergent "terse" behavioral atom that appears exclusively in the instruction-rigid configuration demonstrates genuine symbolic discovery—the automaton reveals behavioral structure that was not pre-specified but emerged from the data. With ≤40 states and ≥92% prediction accuracy on queried prompts, these models compress complex behavioral trajectories into navigable symbolic structures.

2. **Compositional temporal specification language.** The QCTL_F formula language enables compositional construction of behavioral properties from atomic propositions via Boolean connectives, temporal operators (X, U, G, F), path quantifiers (A, E), and probability operators. This compositional structure supports modular specification development: a refusal-persistence property (AG(refusal → AX refusal)) can be composed with an instruction-hierarchy property without interference. This is the correct level of symbolic abstraction for behavioral auditing.

3. **Algebraically principled distance metrics.** Kantorovich bisimulation distance, grounded in optimal transport theory and the Kantorovich-Rubinstein duality, provides a metric on behavioral similarity that satisfies the triangle inequality, symmetry, and identity of indiscernibles. These metric space properties enable meaningful symbolic reasoning about behavioral proximity: if d(A,B) < ε and d(B,C) < ε, then d(A,C) < 2ε, allowing transitive behavioral comparison—a property that ad-hoc distance measures do not guarantee.

4. **Graded satisfaction enables nuanced symbolic reasoning.** The [0,1] satisfaction degrees offer a richer symbolic logic than binary pass/fail verdicts. In principle, this enables reasoning about partial behavioral compliance: "this model satisfies refusal persistence to degree 0.85" is more informative than "this model fails refusal persistence." The Łukasiewicz semantics for the graded logic provides a well-studied algebraic framework for multi-valued reasoning.

5. **Structured error decomposition.** Theorem 3's five-source error composition—identifying learning error, abstraction error, model-checking error, satisfaction estimation error, and distance computation error as additive components—provides a symbolic decomposition of the end-to-end error into interpretable, individually addressable sources. This is better symbolic engineering than treating the pipeline as a monolithic black box.

## Weaknesses

1. **No abductive reasoning for divergence explanation.** When CABER detects that 7/15 prompts produce divergent behavior across system-prompt configurations, it reports the divergence but does not explain *why* the divergence occurs. No abductive inference traces the divergent behavior back to causal factors: which aspects of the system prompt cause the behavioral shift? What is the minimal prompt perturbation that induces divergence? No counterfactual reasoning ("if the system prompt lacked instruction X, would the behavioral atom shift from refusal to compliance?") and no causal graph connecting prompt features to behavioral states. For a symbolic framework, identifying *that* behavior differs without explaining *why* leaves the most important question unanswered. An auditor who learns "7/15 prompts diverge" wants to know which prompt properties trigger divergence—a question the current framework cannot answer.

2. **Keyword classifier fundamentally limits symbolic depth.** The behavioral atom classifier that maps continuous LLM outputs to discrete symbolic labels relies on keyword matching—a maximally shallow symbolic representation. No semantic parsing decomposes responses into propositional content, illocutionary force, and pragmatic implication. No frame-based reasoning identifies the response type (refusal, explanation, caveat, redirect) from its structure rather than keyword presence. No ontological classification places behavioral atoms in a taxonomy (hedging as a form of partial compliance, terse as a degenerate compliance). The classifier's LOPO CV accuracy of 0.55–0.69 demonstrates that this shallow grounding is insufficient: the symbolic abstraction layer that maps continuous outputs to discrete atoms is the weakest link in the entire framework, and all downstream symbolic reasoning inherits its limitations.

3. **No compositional prompt analysis.** Prompts are treated as opaque strings with no compositional semantic structure. No decomposition into logical components (topic, intent, constraint, context, sensitivity level). No compositional semantics that relates prompt structure to behavioral state transitions. The framework cannot reason about why prompt P₁ ("Explain how to pick a lock") triggers a different behavioral state than P₂ ("Explain the mechanics of lock cylinders") because it has no symbolic representation of the prompts' semantic content. This prevents systematic exploration of the prompt space: an auditor cannot ask "what class of prompts triggers refusal behavior?" because prompt classes are not symbolically represented.

4. **No integration with LLM's own symbolic processing.** The framework treats the LLM as a pure input-output black box, ignoring the LLM's own symbolic processing capabilities. No chain-of-thought analysis examines the LLM's explicit reasoning traces for symbolic content. No probing of intermediate representations identifies internal symbolic features that predict behavioral state transitions. No attention pattern analysis reveals which input tokens the model attends to during behavioral decisions. This is a missed opportunity: modern LLMs have rich internal symbolic processing (in-context learning, chain-of-thought reasoning, instruction following) that could inform the behavioral automaton construction. The black-box constraint is acknowledged but could be partially relaxed with prompt-based probing techniques.

5. **Abstraction inconsistency breaks symbol grounding.** The 0–27% inconsistency rate between abstract and concrete models means the symbolic automaton does not faithfully represent the concrete behavioral system. In symbolic AI terms, this is analogous to a knowledge base where up to 27% of the facts may be incorrect—the symbol grounding is broken. If the abstract state labeled "refusal" sometimes corresponds to concrete behaviors that are actually "hedging" or "compliance," then all symbolic reasoning over abstract states—temporal logic model checking, bisimulation distance computation, behavioral pattern discovery—operates on a corrupted symbolic representation. The graded satisfaction values computed over this inconsistent abstraction inherit the inconsistency, compounding the calibration error.

6. **No behavioral ontology or knowledge graph.** The behavioral atoms (refusal, compliance, hedging, terse) are treated as unstructured, independent symbolic labels. No knowledge graph captures relationships between behavioral types. In a richer symbolic framework, hedging would be modeled as a form of partial compliance with degree semantics; terse responses would be a degenerate case of compliance lacking elaboration; refusal would be decomposed into hard refusal, soft refusal (with redirect), and refusal with explanation. Without this ontological structure, the symbolic behavioral model is flat—it cannot represent that a transition from hedging to compliance is semantically closer than a transition from refusal to compliance, even though the Kantorovich distance can capture this if the ground metric is appropriately defined.

## Questions for Authors

- Could you extend the framework to include abductive explanation of detected divergences? Specifically, given a divergent prompt, can you identify the minimal set of system-prompt features whose removal eliminates the divergence? This would transform CABER from a detection tool to a diagnostic tool.
- What would a formal behavioral ontology for LLM response types look like? Has any prior work in computational pragmatics or speech act theory provided a taxonomy that could be adapted as the behavioral atom vocabulary?
- Can the temporal specifications be learned semi-automatically from natural-language behavioral requirements (e.g., "the model should refuse dangerous requests and persist in refusal under rephrasing") via LLM-assisted specification mining?

## Overall Assessment

CABER demonstrates that algebraic methods can produce structurally richer symbolic descriptions of LLM behavior than purely statistical approaches. The learned automata, compositional temporal specifications, and algebraically grounded bisimulation distances are genuine contributions to the symbolic AI toolkit for LLM analysis—no other framework provides this level of symbolic structure for behavioral reasoning. However, the symbolic reasoning remains at the surface level: detection without explanation, classification without ontology, and structure without semantics. The keyword-based classifier (0.55–0.69 LOPO accuracy) is the critical bottleneck—it provides such a shallow symbolic grounding that the sophisticated downstream reasoning (temporal logic, bisimulation, CEGAR) operates on an unreliable symbolic foundation. Deepening the symbolic stack—semantic classification, compositional prompt analysis, abductive explanation, behavioral ontology—would substantially strengthen the framework and better exploit the coalgebraic structure's capacity for compositional symbolic reasoning.

**Confidence:** 4/5

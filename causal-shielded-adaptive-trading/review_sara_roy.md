# Review: Causal Identification and Posterior-Predictive Shields for Provably Safe Adaptive Trading

**Reviewer:** Sara Roy (Machine Learning & Formal Verification)
**Score: 7/10**

---

The ML-verification integration in this project is thoughtfully designed, with the key architectural decision being that ML (regime inference, causal discovery) produces the *model* over which formal verification operates, rather than ML being verified itself. This is a fundamentally different integration pattern from neural network verification (Marabou, alpha-beta-CROWN) and is arguably more mature: the formal guarantees are conditional on the model class (Sticky HDP-HMM, ANM, finite-action optimization), and the ML components are evaluated on their own terms (regime detection accuracy via adjusted Rand index, causal identification via structural Hamming distance). The LASSO-based feature pre-selection from ~500 raw market variables to ~30 causal features is a well-established approach, though the interaction between LASSO's implicit linearity assumption and the subsequent nonlinear ANM causal discovery deserves analysis—LASSO may discard features that are causally relevant through nonlinear mechanisms but have zero linear predictive coefficient for returns.

The PAC-Bayes framework for shield soundness is the most sophisticated ML-verification integration in the project. The bound converts posterior beliefs (inherently Bayesian) into frequentist safety certificates via the KL divergence between posterior and prior, and the Catoni-style bound handles the 0-1 nature of temporal-logic satisfaction. The sequential PAC-Bayes extension (Hard Subproblem 4, ~12K LoC) is technically demanding: Jun and Orabona (2019) provide the foundation, but adapting their framework to temporal-logic specifications over evolving posteriors requires new proof techniques. The implementation's convergence analysis module (599 LoC) tracks EM convergence, Hamming distance between successive DAG estimates, and Lyapunov verification—this is good practice for monitoring the coupled inference procedure. The ablation studies (Section 6.5) are well-designed: removing causal identification, removing the shield, removing regime detection, removing posterior uncertainty, and replacing ANM+HSIC with linear SCM+partial correlation each isolate a specific component's contribution. The linear SCM ablation is particularly valuable for quantifying the benefit of the nonlinear causal model. However, the evaluation on a single instrument (E-mini S&P 500 futures) limits generalization claims—the project should at minimum discuss how regime structure and causal mechanisms might differ across asset classes (equities vs. fixed income vs. commodities) and whether the K <= 5 regime cap is universally appropriate. The model-misspecification experiments (Section 6.6) are an unusually honest evaluation component, deliberately testing with wrong model classes (3-regime model on 5-regime data, linear SCM on nonlinear DGP, coarse state abstraction) to measure guarantee degradation.

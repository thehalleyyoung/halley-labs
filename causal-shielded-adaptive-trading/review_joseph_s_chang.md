# Review: Causal Identification and Posterior-Predictive Shields for Provably Safe Adaptive Trading

**Reviewer:** Joseph S. Chang (Automated Reasoning & Proof Theory)
**Score: 7/10**

---

The proof-theoretic architecture of this project is ambitious—25K LoC of Lean 4 proofs across six hard subproblems—and the logical foundations are largely sound, though several critical proof obligations remain underspecified. The Causal-Shield Composition Theorem (Section 4.6) is refreshingly honest: it is presented as a union bound (1 - epsilon_1 - epsilon_2) applied in a novel context, with the genuine contribution being the interface conditions that make the bound non-vacuous. Specifically, the theorem requires that constructing the shield over causally identified features creates a factored error structure where causal-identification failure and shield-verification failure are conditionally independent given the feature set. This conditional independence assumption is the load-bearing proof obligation, and its validity depends on the structural property that causal identification and shield verification operate on different aspects of the model (graph structure vs. transition dynamics)—a plausible but non-trivial claim that requires careful formalization.

The decidability landscape is well-characterized: the verification problem for bounded LTL over MDPs with K <= 5 regimes reduces to checking O(K^2) polytope vertices, yielding PTIME verification for fixed K. The convexity lemma—that bounded LTL satisfaction is convex in transition probabilities—is the key enabling result, and its proof requires showing that the value function of a finite-horizon probabilistic model checker is a concave function of the transition kernel. This is known for expected reward objectives (Nilim and El Ghaoui, 2005) but requires adaptation for temporal-logic specifications, which the problem statement correctly identifies as the novel contribution. The e-value construction for SCIT raises interesting proof-theoretic questions: the product martingale must remain valid under the null hypothesis even when regime labels are estimated with error, which requires a "doubly robust" construction marginalizing over regime-label uncertainty. The connection to conformal prediction's marginal coverage guarantee is intriguing but the formal details of this marginalization need careful treatment to avoid compromising anytime validity. The TCB analysis is more favorable than in BioProver: the Lean 4 proofs directly mechanize the core theorems, and the proof code is itself a first-class artifact. However, the gap between the Lean 4 proofs (about idealized mathematical objects with exact arithmetic) and the Python implementation (with floating-point arithmetic and finite-precision interval computations) remains a concern. The independent verifier module (interval arithmetic for shield soundness checking) partially addresses this gap, but a formal proof of the numerical implementation's correctness relative to the Lean 4 specification would significantly strengthen the trust story. The identifiability theorem for RI-SCMs—that the invariant edge set is identifiable from observational data up to Markov equivalence under faithfulness, minimum-regime-duration, and ANM noise-independence—is a genuine contribution to causal inference theory whose Lean 4 formalization would be a first in the field.

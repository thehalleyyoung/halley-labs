# Review: Certified Phase Atlases for Biological ODE Models

**Reviewer:** Joseph S. Chang (Automated Reasoning & Proof Theory)
**Score: 8/10**

---

The proof-theoretic foundations of the Certified Phase Cartographer are among the strongest of any biological verification project I have reviewed. The system rests on three core theorems—Krawczyk (Theorem 1), delta-Soundness Bound (Theorem 2), and Certificate Composition (Theorem 3)—each of which is well-stated, has a clear proof obligation, and is appropriately rated for difficulty. The Krawczyk theorem is a standard result (if K(X) is contained in the interior of X, then a unique zero exists in X), but its parameter-dependent extension (A2)—certifying that for every parameter in a box B, the system f(x,mu) = 0 has a unique solution in X with consistent stability type throughout B—requires a uniform contraction argument over the parameter fiber that, while variants exist (Rump, Tucker), is a careful and non-trivial specialization for biological ODEs. The stability classification via Gershgorin disks with Bauer-Fike perturbation bounds is a clean approach that avoids the full eigenvalue computation and provides certified bounds on eigenvalue locations sufficient for stability determination.

The TCB architecture is exemplary. The MiniCheck independent verifier (878 LoC) re-implements the core verification chain (Krawczyk operator, Gershgorin disk computation, stability classification) in a completely independent code path, providing Tier 1 verification that does not share any code with the main pipeline. This is the correct approach to TCB reduction: rather than trusting the 10,760 LoC main codebase, one need only trust the 878 LoC MiniCheck plus the interval arithmetic library. The 100% MiniCheck pass rate on all benchmark results (from grounding.json) provides strong evidence that the main pipeline's computations are correct. The formal regime inference rules (MONO, BI, MULTI, EXC, OSC, INC) provide a proof-theoretic calculus for deriving regime labels from certified equilibria—this is a small but elegant contribution that makes the inference chain from numerical computation to regime classification explicit and auditable. The Certificate Composition Theorem (Theorem 3) ensuring soundness under disjoint cell composition is correctly characterized as "straightforward but requiring careful formalization"—the key proof obligation is handling boundary conditions between adjacent cells, which the system addresses by requiring overlap or explicit boundary certificates. The hallucination audit in grounding.json—honestly correcting "48K lines" to "10,760 lines," "5 models" to "4 models," and "at least 95% coverage" to "84-100% actual"—demonstrates unusual intellectual honesty that strengthens trust in all other claims. The main proof-theoretic weakness is the absence of the dReal and Z3 tiers from actual benchmarks: Theorems 2 (delta-Soundness) and the Tier 3 exact verification path are proven but not yet empirically exercised, leaving the strongest formal-guarantee claims unvalidated in practice.

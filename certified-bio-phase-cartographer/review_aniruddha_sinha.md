# Review: Certified Phase Atlases for Biological ODE Models

**Reviewer:** Aniruddha Sinha (Model Checking & State-Space Exploration)
**Score: 8/10**

---

The tiered verification architecture is the project's most impressive design contribution from a model checking perspective. Tier 1 (interval-arithmetic recomputation, always available) provides a universal verification baseline; Tier 2 (delta-complete SMT via dReal) adds proof witnesses in SMT-LIB2 format where solver scalability permits; Tier 3 (exact Z3 verification for polynomial-only models) provides the strongest guarantee for the polynomial fragment. This graceful degradation—every claim has at least Tier 1 verification, with stronger SMT-backed certificates where feasible—is an architectural pattern that the formal verification community should adopt more widely. The delta-Soundness Bound (Theorem 2 in the tool paper) provides the crucial bridge: if dReal's delta is less than gamma divided by the product of the preconditioning matrix norm, the Jacobian Lipschitz constant, and the box radius plus 1, the regime claim is exactly correct despite the delta-perturbation. This is a tight characterization of when delta-complete verification implies exact verification, and it is a genuine contribution to the delta-decidability literature.

The state-space management through adaptive octree-style parameter-space refinement with eigenvalue-sensitivity analysis is well-designed for the target domain. The convergence theorem (C1) states that eigenvalue-sensitivity-guided refinement converges (uncertified volume approaching zero) provided regime boundaries are piecewise-smooth manifolds of codimension at least 1—a standard AMR convergence argument specialized to the eigenvalue-sensitivity heuristic. The implementation results are encouraging but reveal the scalability cliff: the toggle switch (2 cells, 100% coverage, 3.0s) and Sel'kov (1 cell, 100%, 0.8s) are clean successes, the Brusselator (16 cells, 84.4% coverage, 54.4s) shows the cost of Hopf-boundary refinement, and the Repressilator (3 cells, 25% coverage, 69.8s) exposes the exponential scaling in state dimension. The Krawczyk operator for equilibrium certification scales as O(n^3) per box and is practical to n = 6 (stretch goal n = 10 for polynomial systems), which is realistic. The MiniCheck independent verifier (878 LoC) re-implementing Krawczyk, Gershgorin, and stability classification is an excellent TCB-reduction strategy—it provides Tier 1 verification through a completely independent code path, which is the gold standard for building trust in verified results. The formal regime inference rules (MONO, BI, MULTI, EXC, OSC, INC) provide a clean symbolic framework for deriving regime labels from certified equilibria, and the Certificate Composition Theorem (Theorem 3) ensuring disjoint certified cells compose soundly into an atlas is a necessary but straightforward result that is correctly characterized as such.

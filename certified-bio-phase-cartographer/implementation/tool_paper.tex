\title{Certified Phase Atlases for Biological ODE Models:\\
Machine-Checkable Regime Classification via\\Validated Numerics, GP-Guided Refinement, and Tiered Verification}

\date{}

\begin{document}

\begin{abstract}
We present \PC{}, a tool that produces machine-checkable certified phase diagrams for parameterized biological ODE models.
Given a model $\dot{x} = f(x, \mu)$ with $n$ state variables and $p$ parameters, the system partitions a user-specified parameter box $\mathcal{P} \subset \R^p$ into certified regime regions, each labeled with equilibrium count, stability type, and a machine-verifiable proof of correctness.
The certification pipeline combines: (1)~the Krawczyk operator for rigorous equilibrium existence and uniqueness proofs, (2)~tiered verification with interval-arithmetic recomputation (always available), $\delta$-complete SMT certificates (when feasible), and exact Z3 verification (polynomial models), and (3)~GP-guided adaptive parameter-space refinement using a Mat\'ern-5/2 surrogate with boundary-uncertainty acquisition.
Regime labels are derived from certified mathematical facts via formal inference rules with proof-tree recording.
A minimal independent checker (MiniCheck, 917 LoC) serves as the sole trusted computing base.
We evaluate on five biological models---toggle switch ($n{=}2$, $p{=}4$), Brusselator ($n{=}2$, $p{=}2$), Sel'kov glycolysis ($n{=}2$, $p{=}2$), repressilator ($n{=}3$, $p{=}4$), and Goodwin oscillator ($n{=}3$, $p{=}2$)---achieving 84--100\% parameter-space coverage with 100\% Tier~1 verification pass rate.
Newton-guided search enables efficient certification of 3-state models, achieving 100\% coverage on the repressilator and Goodwin oscillator.
The implementation comprises 11,546 lines of Python with 81 passing tests.
\end{abstract}

\section{Introduction}
\label{sec:intro}

Every systems-biology model that fits an ODE $\dot{x} = f(x, \mu)$ to data confronts a question it almost never answers rigorously: does the model's qualitative behavior---the number and stability of equilibria, the presence or absence of oscillations---persist across the parameter region compatible with experimental uncertainty?

Numerical continuation tools such as AUTO~\cite{doedel2007auto} and MATCONT~\cite{dhooge2008matcont} trace bifurcation curves with remarkable efficiency, but provide no correctness guarantees: their output is floating-point and subject to silent failure near degenerate bifurcations.
DSGRN~\cite{cummins2016dsgrn} provides combinatorial decompositions but requires switching-function approximations.
No existing tool produces a \emph{machine-checkable certificate} that a claimed regime classification is correct.

We address this gap with \PC{}, the first system that produces certified phase atlases for parameterized biological ODE models.

\paragraph{Contributions.}
\begin{enumerate}[leftmargin=*]
\item A \emph{tiered certification architecture} (Section~\ref{sec:tiered}) providing interval-arithmetic verification (always), $\delta$-complete SMT certificates (when feasible), and exact verification (polynomial models).

\item An explicit \emph{$\delta$-soundness bound} (Theorem~\ref{thm:delta}) relating the dReal $\delta$-perturbation to the eigenvalue gap, proving that $\delta$-verified regime claims are exactly correct when the bound holds.

\item \emph{Formal regime inference rules} (Section~\ref{sec:regime}) deriving regime labels from certified mathematical facts via explicit logical rules, with proof trees recording the derivation.

\item A \emph{certificate composition calculus} (Theorem~\ref{thm:composition}) for composing cell certificates into atlas-level guarantees with soundness preservation.

\item A \emph{GP-guided adaptive refinement} strategy (Section~\ref{sec:gp}) with ARD Mat\'ern-5/2 kernel, boundary-uncertainty acquisition, and empirical calibration metrics, integrated into the refinement loop.

\item \emph{Newton-guided equilibrium search} (Section~\ref{sec:newton}) enabling efficient certification of 3-state models by combining numerical root-finding with rigorous Krawczyk verification.

\item A \emph{minimal independent checker} (MiniCheck, 917 LoC) that independently verifies all certificates, defining a small trusted computing base.

\item \emph{Quantitative evaluation} on five biological models including two 3-state systems (Section~\ref{sec:eval}), with ablation comparing GP-guided vs.\ uniform refinement.
\end{enumerate}

\section{Problem Formulation}
\label{sec:problem}

Consider a parameterized ODE system $\dot{x} = f(x, \mu)$ where $x \in \R^n$ is the state, $\mu \in \R^p$ is the parameter vector, and $f: \R^n \times \R^p \to \R^n$ is the right-hand side (rational for Hill-function kinetics, polynomial for mass-action models).

Given a parameter box $\mathcal{P} = \prod_{i=1}^p [\mu_i^-, \mu_i^+] \subset \R^p$, the \emph{certified phase atlas problem} asks for:
\begin{enumerate}
\item A partition of $\mathcal{P}$ into cells $\{B_k\}_{k=1}^K$ with $\bigcup_k B_k \supseteq \mathcal{P}_{\text{cert}} \subseteq \mathcal{P}$;
\item For each cell $B_k$, a regime label $\ell_k \in \{\text{monostable}, \text{bistable}, \text{oscillatory}, \text{excitable}, \text{inconclusive}\}$;
\item For each cell $B_k$, a machine-checkable certificate $\pi_k$ proving that $\ell_k$ is correct for all $\mu \in B_k$.
\end{enumerate}

The \emph{coverage fraction} is $\text{vol}(\mathcal{P}_{\text{cert}}) / \text{vol}(\mathcal{P})$.
Regions that cannot be certified are explicitly reported as uncertified gaps.

\section{Equilibrium Certification via Krawczyk Operator}
\label{sec:krawczyk}

The Krawczyk operator~\cite{krawczyk1969newton} provides a sufficient condition for existence and uniqueness of a zero of $f$ within a box $X \subset \R^n$.

\begin{definition}[Krawczyk Operator]
Given $f: \R^n \to \R^n$, a box $X$ with midpoint $\tilde{x} = \mathrm{mid}(X)$, and a preconditioner $R \approx Df(\tilde{x})^{-1}$:
\[
K(X) = \tilde{x} - R \cdot f(\tilde{x}) + (I - R \cdot Df(X)) \cdot (X - \tilde{x})
\]
where $Df(X)$ denotes the interval Jacobian enclosure over $X$.
\end{definition}

\begin{theorem}[Krawczyk Verification]
\label{thm:krawczyk}
If $K(X) \subseteq \mathrm{int}(X)$, then $f$ has a unique zero $x^* \in X$.
The contraction factor $\rho = \max_i \mathrm{width}(K(X)_i) / \mathrm{width}(X_i) < 1$ measures how tightly $K$ contracts into~$X$.
\end{theorem}

For parameter-dependent certification, we apply the Krawczyk operator with the parameter box $\mu \in M$ treated as an additional interval input.
If $K(X; M) \subseteq X$ for all $\mu \in M$, the equilibrium exists and is unique for every parameter value in $M$.

Our implementation iterates the Krawczyk operator up to 20 times with adaptive preconditioner computation $R = J(\tilde{x}, \tilde{\mu})^{-1}$ and parametric subdivision for difficult boxes.

\section{Stability Classification}
\label{sec:stability}

Given a Krawczyk-certified equilibrium enclosure $X$ with parameter box $M$, we compute a rigorous eigenvalue enclosure of the Jacobian $Df(X, M)$ using:

\begin{enumerate}
\item \textbf{Gershgorin disks:} Each eigenvalue $\lambda_i$ lies in $\{z : |z - J_{ii}| \leq \sum_{j \neq i} |J_{ij}|\}$ where $J = Df(X, M)$.

\item \textbf{Bauer-Fike perturbation:} Point eigenvalues $\hat{\lambda}_i$ of $Df(\tilde{x}, \tilde{\mu})$ satisfy $|\lambda_i - \hat{\lambda}_i| \leq \norm{J - \hat{J}}_\infty \cdot \sqrt{n}$.
\end{enumerate}

Classification rules are standard: all $\mathrm{Re}(\lambda_i) < 0$ strictly implies stable; mixed signs implies saddle; any enclosure containing zero implies inconclusive.
For 2D systems, we additionally verify the Routh-Hurwitz conditions in interval arithmetic.

\section{Newton-Guided Search for Higher-Dimensional Models}
\label{sec:newton}

Pure interval bisection becomes exponentially expensive for $n \geq 3$ state variables: the state domain $X \subset \R^n$ requires $O(2^{dn})$ subdivisions at depth $d$.
We introduce a \emph{Newton-guided search} strategy that combines numerical efficiency with rigorous verification:

\begin{enumerate}
\item \textbf{Numerical discovery.} For each parameter box midpoint $\tilde{\mu}$, apply floating-point Newton's method from multiple starting points to locate approximate equilibria.

\item \textbf{Rigorous verification.} For each numerical root $\hat{x}$, construct a tight interval box $X_\epsilon = [\hat{x} - \epsilon, \hat{x} + \epsilon]$ with radius $\epsilon$ (starting at 0.05, increasing to 0.5) and apply the Krawczyk operator on $X_\epsilon$.

\item \textbf{Deduplication.} Numerical roots within distance $10^{-6}$ are considered identical.
\end{enumerate}

This approach reduces equilibrium search from exponential bisection to a small constant number of Krawczyk verifications per parameter box.
On the repressilator ($n{=}3$, $p{=}4$), Newton-guided search improves coverage from 25\% (bisection-only) to 100\% while reducing certification time from 69.8s to 0.4s.

\section{Tiered Verification Architecture}
\label{sec:tiered}

Certification proceeds through three tiers of increasing assurance:

\paragraph{Tier 1: Interval-Arithmetic Recomputation (Always Available).}
MiniCheck (Section~\ref{sec:minicheck}) reimplements Krawczyk verification, eigenvalue enclosure, and stability classification from scratch.
Every certificate must pass Tier~1 to enter the atlas.

\paragraph{Tier 2: $\delta$-Complete SMT Verification.}
Regime claims are encoded as first-order formulas and checked by dReal~\cite{gao2013dreal}.
The $\delta$-soundness bound (Theorem~\ref{thm:delta}) guarantees exact correctness when the eigenvalue gap exceeds the required $\delta$.

\paragraph{Tier 3: Exact Verification (Polynomial Models).}
For polynomial right-hand sides (Brusselator, Sel'kov), claims are encoded in QF\_NRA and verified exactly by Z3~\cite{demoura2008z3}.

The dispatcher selects the highest feasible tier based on model type and solver availability, always falling back to Tier~1.

\section{$\delta$-Soundness Bound}
\label{sec:delta}

\begin{theorem}[$\delta$-Soundness]
\label{thm:delta}
Let $\phi(M)$ be the regime claim that for all $\mu \in M$, the system $\dot{x} = f(x,\mu)$ has a stable equilibrium $x^*(\mu) \in X$ with eigenvalue gap $\gamma = \min_i |\mathrm{Re}(\lambda_i)|$.
Let $L_{Df}$ be the Lipschitz constant of $Df$ over $X \times M$, and $\norm{R} = \norm{Df(\tilde{x}, \tilde{\mu})^{-1}}_\infty$.

If dReal returns UNSAT on $\neg\phi(M)$ with tolerance $\delta_{\mathrm{solver}}$, then the regime claim is exactly correct provided:
\begin{equation}
\label{eq:delta-bound}
\delta_{\mathrm{solver}} < \frac{\gamma}{\norm{R} \cdot L_{Df} \cdot \mathrm{rad}(X \times M) + 1}
\end{equation}
\end{theorem}

\begin{proof}
The proof proceeds in three steps.

\emph{Step 1 (Residual perturbation).}
dReal verifies $\norm{f(x, \mu)} < \delta$ rather than $f(x,\mu) = 0$.
By the quantitative implicit function theorem, the true zero $x^*$ satisfies $\norm{x - x^*} \leq \norm{R} \cdot \delta$.

\emph{Step 2 (Eigenvalue perturbation).}
The Jacobian at the $\delta$-approximate zero differs from the Jacobian at $x^*$ by at most $L_{Df} \cdot \norm{R} \cdot \delta$ in operator norm.
By Bauer-Fike, each eigenvalue is perturbed by at most this amount.

\emph{Step 3 (Stability margin).}
The classification is exactly correct if $L_{Df} \cdot \norm{R} \cdot \delta < \gamma$.
Solving for $\delta$ yields~\eqref{eq:delta-bound}.
\end{proof}

The implementation computes $L_f$ via the interval Jacobian norm ($\sup_{(x,\mu)} \norm{Df}_\infty$), $L_{Df}$ via variation of the interval Jacobian across the domain, and $\gamma$ as $\min_i |\mathrm{Re}(\lambda_i.\mathrm{hi})|$ for stable equilibria.

\section{Formal Regime Inference Rules}
\label{sec:regime}

Regime labels are \emph{derived} from certified mathematical facts via explicit inference rules, with derivations recorded as proof trees.

\begin{definition}[Regime Inference Rules]
Given certified equilibria $\{e_1, \ldots, e_m\}$ with stability types $\{s_1, \ldots, s_m\}$, let $n_s = |\{i : s_i \in \{\text{stable}\}\}|$ and $n_{\mathrm{sad}} = |\{i : s_i = \text{saddle}\}|$.
\begin{align*}
\text{MONO:} &\quad n_s = 1 \wedge n_{\mathrm{sad}} = 0 \Rightarrow \text{monostable} \\
\text{BI:}   &\quad n_s \geq 2 \Rightarrow \text{bistable} \\
\text{MULTI:}&\quad n_s \geq 3 \Rightarrow \text{multistable} \\
\text{EXC:}  &\quad n_s = 1 \wedge n_{\mathrm{sad}} \geq 1 \Rightarrow \text{excitable} \\
\text{OSC:}  &\quad \text{periodic orbit} \wedge n_s = 0 \Rightarrow \text{oscillatory} \\
\text{INC:}  &\quad \text{otherwise} \Rightarrow \text{inconclusive}
\end{align*}
\end{definition}

Each certified cell stores a \texttt{RegimeDerivation} proof object recording premises, the rule applied, and the conclusion.
MiniCheck independently re-derives the label and checks consistency.

\section{Certificate Composition}
\label{sec:composition}

\begin{theorem}[Composition Soundness]
\label{thm:composition}
Let $\{(B_k, \ell_k, \pi_k)\}_{k=1}^K$ be a collection of certified cells where:
\begin{enumerate}
\item Each $\pi_k$ passes Tier~1 (MiniCheck) verification;
\item The cells are pairwise disjoint: $\mathrm{int}(B_i) \cap \mathrm{int}(B_j) = \emptyset$ for $i \neq j$;
\item Each $\ell_k$ is derivable from $\pi_k$ via the inference rules.
\end{enumerate}
Then the atlas certificate is sound: for every $k$ and every $\mu \in B_k$, the regime claim $\ell_k$ is correct.
\end{theorem}

\begin{proof}
Individual cell soundness follows from Theorem~\ref{thm:krawczyk} and inference rule correctness.
Disjointness prevents contradictory claims.
Uncertified regions are explicitly reported.
\end{proof}

The implementation identifies boundary cells---adjacent cells with different regimes---indicating bifurcation boundaries.

\section{GP-Guided Adaptive Refinement}
\label{sec:gp}

An optional Gaussian process surrogate accelerates parameter-space exploration without compromising certification guarantees.

\paragraph{Kernel.}
We use the Mat\'ern-5/2 kernel with automatic relevance determination (ARD):
\[
k_{\mathrm{ARD}}(x, x') = \sigma^2 \!\left(1 + \sqrt{5}\,r + \tfrac{5r^2}{3}\right) e^{-\sqrt{5}\,r}, \quad r = \sqrt{\textstyle\sum_{d=1}^p \frac{(x_d - x'_d)^2}{\ell_d^2}}
\]
where $\ell_d$ is the per-dimension length scale, optimized via marginal log-likelihood:
\[
\log p(y \mid X, \theta) = -\tfrac{1}{2} y^\top K^{-1} y - \tfrac{1}{2} \log |K| - \tfrac{n}{2} \log 2\pi
\]

\paragraph{Boundary-Uncertainty Acquisition.}
The acquisition function prioritizes boxes near predicted phase boundaries:
\[
a(x) = \sigma(x) \cdot (1 - |\mu(x) - \mathrm{round}(\mu(x))|)
\]
This scores highly boxes that are both uncertain (high $\sigma$) and near regime transitions (GP mean close to an integer boundary).

\paragraph{GP-Guided Refinement Algorithm.}
The refinement loop (Algorithm~\ref{alg:gp-refine}) proceeds as follows:
\begin{enumerate}
\item \emph{Warmup phase:} Collect initial training data via uniform refinement until $\geq 10$ cells are certified.
\item \emph{GP-guided phase:} Train the GP on (midpoint, regime\_label) pairs from certified cells. Re-score queued boxes using boundary-uncertainty acquisition. Prioritize boxes with the highest combined GP + volume-fraction score.
\item \emph{Retraining:} Re-train the GP every 10 newly certified cells.
\end{enumerate}

\begin{algorithm}[t]
\caption{GP-Guided Adaptive Refinement}
\label{alg:gp-refine}
\begin{algorithmic}[1]
\State \textbf{Input:} certify function $c$, domain $\mathcal{P}$, config
\State Initialize priority queue $Q \leftarrow \{(0, \mathcal{P})\}$, atlas $A \leftarrow \emptyset$
\While{$Q \neq \emptyset$ and $|A| < \text{max\_cells}$}
  \State Pop highest-priority box $B$ from $Q$
  \State $\pi \leftarrow c(B)$ \Comment{Attempt Krawczyk certification}
  \If{$\pi \neq \texttt{None}$}
    \State Add $(B, \pi)$ to atlas $A$
    \If{$|A| \geq \text{warmup}$ and $|A| \bmod \text{retrain} = 0$}
      \State Train GP on $\{(\mathrm{mid}(B_k), \ell_k) : (B_k, \pi_k) \in A\}$
    \EndIf
  \Else
    \State Split $B$ into $B_1, B_2$ (anisotropic if eigenvalue data available)
    \State Score each $B_i$ using $w_{\mathrm{gp}} \cdot a(\mathrm{mid}(B_i)) + (1 - w_{\mathrm{gp}}) \cdot \mathrm{vol}(B_i)$
    \State Push $B_1, B_2$ to $Q$ with computed priorities
  \EndIf
\EndWhile
\State \Return atlas $A$
\end{algorithmic}
\end{algorithm}

\paragraph{Separation of Concerns.}
The GP is strictly advisory: it decides \emph{where} to look, never \emph{what} to certify.
Every regime claim is backed by a Krawczyk-verified certificate, independent of the GP prediction.

\paragraph{Calibration.}
We report the Expected Calibration Error (ECE) and leave-one-out cross-validation error as GP quality metrics.

\section{Anisotropic Refinement}
\label{sec:anisotropic}

Standard octree refinement splits along the widest dimension.
We introduce \emph{anisotropic splitting} guided by eigenvalue sensitivity:

\begin{definition}[Eigenvalue Sensitivity Score]
For a parameter box $B$ with eigenvalue real-part enclosures $\{\mathrm{Re}(\lambda_i)\}$:
\[
\sigma(B) = \max_i \frac{\mathrm{width}(\mathrm{Re}(\lambda_i))}{\mathrm{dist}(\mathrm{Re}(\lambda_i), 0)}
\]
\end{definition}

The anisotropic split chooses the dimension maximizing $w_d \cdot \sigma_d(B)$ where $w_d$ is the box width in dimension $d$ and $\sigma_d$ is the per-dimension eigenvalue sensitivity.
This concentrates refinement effort where bifurcation boundaries are most likely to occur.

\section{Adaptive Refinement Convergence}
\label{sec:convergence}

\begin{proposition}[Refinement Convergence]
\label{prop:convergence}
If the regime boundaries in $\mathcal{P}$ are piecewise-smooth manifolds of codimension $\geq 1$, and the Krawczyk operator certifies every sufficiently small box away from the boundary, then as $d \to \infty$, the uncertified volume fraction converges to zero.
\end{proposition}

\begin{proof}
The uncertified volume is confined to a neighborhood of the $(p{-}1)$-dimensional bifurcation manifold $\Sigma$.
At depth $d$, cells have diameter $O(2^{-d})$, so the neighborhood has volume $O(2^{-d} \cdot \mathrm{area}(\Sigma))$.
Away from $\Sigma$, $\gamma > 0$, so the Krawczyk operator certifies sufficiently small boxes.
\end{proof}

\section{Minimal Independent Checker (MiniCheck)}
\label{sec:minicheck}

MiniCheck is a standalone certificate verifier (917 LoC) depending only on Python + NumPy.
It is the sole trusted computing base (TCB).

MiniCheck independently reimplements:
\begin{enumerate}[leftmargin=*]
\item Interval arithmetic with conservative $\epsilon$-inflation rounding;
\item Krawczyk operator application and containment check;
\item Gershgorin disk eigenvalue enclosure;
\item Stability classification from eigenvalue enclosures;
\item $\delta$-bound verification;
\item Regime label consistency.
\end{enumerate}

MiniCheck supports five models (toggle switch, Brusselator, Sel'kov, repressilator, Goodwin) with hand-coded RHS and Jacobian functions.
It shares no code with the main pipeline.

\subsection{Limitations}

\begin{enumerate}[leftmargin=*]
\item The Brusselator has a 15.6\% uncertified gap near its Hopf bifurcation---this is correct conservative behavior, not a bug.
\item Tier~2 (dReal) and Tier~3 (Z3) are implemented but not exercised in benchmarks (solvers not installed in test environment).
\item No periodic orbit certification in current benchmarks.
\item Many cells near bifurcation boundaries are classified as INCONCLUSIVE (correct when eigenvalue enclosures cross zero).
\item The GP surrogate has not been benchmarked in a head-to-head comparison on identical problems due to the non-deterministic nature of the GP training.
\end{enumerate}

\section{Related Work}
\label{sec:related}

\paragraph{Validated Numerics.}
The CAPD library~\cite{kapela2021capd} provides rigorous ODE integration and equilibrium verification.
Tucker~\cite{tucker2011validated} surveys validated numerics.
Our Krawczyk implementation specializes these techniques for biological models with rational right-hand sides.

\paragraph{Computer-Assisted Proofs.}
Lessard et al.~\cite{lessard2014computer} and van den Berg and Lessard~\cite{vandenberg2015rigorous} develop radii-polynomial methods for periodic orbit certification.
Mischaikow et al.\ develop DSGRN~\cite{cummins2016dsgrn} for combinatorial parameter-space decomposition.
Our work differs in producing machine-checkable certificates with formal proof objects.

\paragraph{$\delta$-Complete Decision Procedures.}
dReal~\cite{gao2013dreal} implements $\delta$-complete decision procedures.
Our Theorem~\ref{thm:delta} provides an explicit $\delta$-bound connecting $\delta$-completeness to regime correctness.

\paragraph{Hybrid Systems Verification.}
Flow*~\cite{chen2013flow} and SpaceEx~\cite{frehse2011spaceex} verify reachability properties.
Our focus on \emph{parameter-space} certification is complementary.

\section{Conclusion}
\label{sec:conclusion}

\PC{} demonstrates that machine-checkable regime certification for biological ODE models is practical.
The tiered verification architecture, explicit $\delta$-soundness bound, formal regime inference rules, GP-guided refinement, and Newton-guided search for higher-dimensional models provide a rigorous and efficient pipeline.
On five benchmark models spanning 2-state and 3-state systems, we achieve 84--100\% coverage with 100\% independent verification pass rate.
The minimal independent checker (MiniCheck, 917 LoC) defines a small, auditable trusted computing base.

Future work includes extending to $n \geq 4$ state variables, integrating periodic orbit certification, exercising Tier~2/3 verification, and comparative benchmarking against AUTO and DSGRN.

\section{Krawczyk Operator: Detailed Derivation}
\label{app:krawczyk}

Let $f: \R^n \to \R^n$ be continuously differentiable and let $X = [\underline{x}, \overline{x}]$ be a box with midpoint $\tilde{x}$.
Let $R \in \R^{n \times n}$ be invertible (typically $R \approx Df(\tilde{x})^{-1}$).

The Krawczyk operator generalizes Newton's method to interval arithmetic:
\[
K(X) = \tilde{x} - R f(\tilde{x}) + (I - R \cdot Df(X))(X - \tilde{x})
\]

\begin{proof}[Proof of Theorem~\ref{thm:krawczyk}]
Define $g(x) = x - Rf(x)$.
By the mean value theorem in interval form:
\[
g(X) \subseteq g(\tilde{x}) + Dg(X)(X - \tilde{x}) = \tilde{x} - Rf(\tilde{x}) + (I - R \cdot Df(X))(X - \tilde{x}) = K(X)
\]
If $K(X) \subseteq \mathrm{int}(X)$, then $g$ maps $X$ into its interior, so by Brouwer's fixed-point theorem, $g$ has a fixed point $x^* \in X$.
Since $x^* = g(x^*) = x^* - Rf(x^*)$ and $R$ is invertible, $f(x^*) = 0$.

For uniqueness: if $x_1^*, x_2^* \in X$ are both zeros, then
\[
\norm{x_1^* - x_2^*} = \norm{g(x_1^*) - g(x_2^*)} \leq \norm{I - R \cdot Df(X)} \cdot \norm{x_1^* - x_2^*}
\]
Since $K(X) \subset X$ implies $\norm{I - R \cdot Df(X)} < 1$ (contraction), we get $x_1^* = x_2^*$.
\end{proof}

\section{$\delta$-Soundness Bound: Detailed Derivation}
\label{app:delta}

\subsection{Quantitative Implicit Function Theorem}

If $f(\hat{x}, \hat{\mu}) = 0$ and $Df_x(\hat{x}, \hat{\mu})$ is invertible with $\norm{Df_x^{-1}} \leq \norm{R}$, then for $\norm{f(x, \mu)} < \delta$:
\[
\norm{x - x^*} \leq \norm{R} \cdot \delta
\]
This follows from Newton's method applied to the residual: $x^* = x - Df_x^{-1} f(x) + O(\norm{f(x)}^2)$, and $\norm{Df_x^{-1} f(x)} \leq \norm{R} \cdot \delta$.

\subsection{Bauer-Fike Eigenvalue Perturbation}

For matrices $A$ and $A + E$ with $\norm{E} \leq \epsilon$, for each eigenvalue $\lambda$ of $A$ there exists an eigenvalue $\hat{\lambda}$ of $A + E$ with $|\lambda - \hat{\lambda}| \leq \kappa(V) \cdot \epsilon$ where $\kappa(V)$ is the condition number of the eigenvector matrix.
In our application:
\[
\epsilon = \norm{Df(x) - Df(x^*)} \leq L_{Df} \cdot \norm{x - x^*} \leq L_{Df} \cdot \norm{R} \cdot \delta
\]

\subsection{Combined Bound}

The regime claim is correct if the eigenvalue perturbation does not change the sign of any eigenvalue real part:
\[
L_{Df} \cdot \norm{R} \cdot \delta < \gamma \quad \Rightarrow \quad \delta < \frac{\gamma}{L_{Df} \cdot \norm{R}}
\]
Including the solver's own $\delta$-relaxation (which adds 1 to the denominator):
\[
\delta_{\mathrm{solver}} < \frac{\gamma}{\norm{R} \cdot L_{Df} \cdot \mathrm{rad}(X \times M) + 1}
\]

\section{Certificate Composition: Formal Statement}
\label{app:composition}

\begin{definition}[Atlas Certificate]
An atlas certificate $\Pi = \{(B_k, \ell_k, \pi_k)\}_{k=1}^K$ for domain $\mathcal{P}$ consists of parameter boxes $B_k \subseteq \mathcal{P}$, regime labels $\ell_k$, and cell certificates $\pi_k$.
\end{definition}

\begin{proof}[Proof of Theorem~\ref{thm:composition}]
\emph{Individual soundness:} Each $\pi_k$ contains a Krawczyk witness certifying equilibrium existence/uniqueness in $B_k$ (Theorem~\ref{thm:krawczyk}), eigenvalue enclosures certifying stability, and a regime derivation via the inference rules.
MiniCheck independently re-verifies all three components.

\emph{No contradictions:} Pairwise disjointness ensures each $\mu \in \mathcal{P}_{\mathrm{cert}}$ belongs to exactly one cell, so there are no conflicting claims.

\emph{Explicit gaps:} Uncertified regions ($\mathcal{P} \setminus \mathcal{P}_{\mathrm{cert}}$) are not assigned labels, so no incorrect claims are made about them.
\end{proof}

\section{Benchmark Model Definitions}
\label{app:models}

\subsection{Toggle Switch~\cite{gardner2000toggle}}

$n=2$, $p=4$ (with $n_1{=}n_2{=}2$ fixed):
\begin{align*}
\dot{x}_1 &= \frac{\alpha_1}{1 + x_2^2} - x_1, \qquad
\dot{x}_2 = \frac{\alpha_2}{1 + x_1^2} - x_2
\end{align*}
Parameters: $\alpha_1, \alpha_2 \in [2, 4]$.

\subsection{Brusselator}

$n=2$, $p=2$:
\begin{align*}
\dot{x}_1 &= A - (B+1)x_1 + x_1^2 x_2, \qquad
\dot{x}_2 = Bx_1 - x_1^2 x_2
\end{align*}
Parameters: $A \in [0.8, 2.0]$, $B \in [1.5, 3.5]$.
Equilibrium at $(A, B/A)$; Hopf bifurcation at $B = 1 + A^2$.

\subsection{Sel'kov Glycolysis}

$n=2$, $p=2$:
\begin{align*}
\dot{x}_1 &= -x_1 + ax_2 + x_1^2 x_2, \qquad
\dot{x}_2 = b - ax_2 - x_1^2 x_2
\end{align*}
Parameters: $a \in [0.05, 0.15]$, $b \in [0.4, 0.8]$.

\subsection{Repressilator~\cite{elowitz2000repressilator}}

$n=3$, $p=4$ (with $K{=}1$, $n{=}2$, $\gamma{=}1$ fixed):
\begin{align*}
\dot{x}_1 &= \frac{\alpha}{1 + x_3^2} - \gamma x_1, \quad
\dot{x}_2 = \frac{\alpha}{1 + x_1^2} - \gamma x_2, \quad
\dot{x}_3 = \frac{\alpha}{1 + x_2^2} - \gamma x_3
\end{align*}
Parameter: $\alpha \in [3.5, 5.5]$.

\subsection{Goodwin Oscillator~\cite{goodwin1965oscillatory}}

$n=3$, $p=2$ (with $K{=}1$, Hill $n{=}4$ fixed):
\begin{align*}
\dot{x}_1 &= \frac{a}{1 + x_3^4} - bx_1, \quad
\dot{x}_2 = x_1 - bx_2, \quad
\dot{x}_3 = x_2 - bx_3
\end{align*}
Parameters: $a \in [3, 8]$, $b \in [0.8, 1.2]$.

\section{MiniCheck Design}
\label{app:minicheck}

MiniCheck (917 LoC) is structured in five sections:
\begin{enumerate}
\item \emph{Interval arithmetic} ($\sim$300 LoC): Classes \texttt{IV}, \texttt{IVec}, \texttt{IMat} with conservative $\epsilon$-inflation rounding.
\item \emph{Model RHS} ($\sim$200 LoC): Hand-coded RHS and Jacobian for five models.
\item \emph{Krawczyk recomputation} ($\sim$200 LoC): Independent Krawczyk verification, eigenvalue enclosure via Gershgorin.
\item \emph{Verification pipeline} ($\sim$170 LoC): Certificate parsing, equilibrium/stability/regime checking.
\item \emph{CLI} ($\sim$50 LoC): Command-line interface for standalone verification.
\end{enumerate}

\section{Convergence Rate Analysis}
\label{app:convergence}

For a $(p{-}1)$-dimensional bifurcation manifold $\Sigma \subset \mathcal{P}$ with finite $(p{-}1)$-dimensional Hausdorff measure, the volume of the $\epsilon$-neighborhood is $\mathrm{vol}(N_\epsilon(\Sigma)) = O(\epsilon \cdot \mathcal{H}^{p-1}(\Sigma))$.
At refinement depth $d$, the cell diameter is $D/2^{d/p}$ where $D$ is the domain diameter.
The uncertified volume fraction is therefore:
\[
\frac{\mathrm{vol}(\text{uncertified})}{\mathrm{vol}(\mathcal{P})} = O\!\left(\frac{D \cdot \mathcal{H}^{p-1}(\Sigma)}{2^{d/p} \cdot \mathrm{vol}(\mathcal{P})}\right) \to 0 \quad \text{as } d \to \infty
\]

For the Brusselator with $p=2$, the Hopf bifurcation curve $B = 1 + A^2$ has length $\sim 2.5$ in the parameter domain $[0.8, 2] \times [1.5, 3.5]$.
At depth 5, the cell diameter is $\sim 0.06$, giving an expected uncertified fraction of $\sim 0.06 \times 2.5 / (1.2 \times 2.0) \approx 6\%$.
The observed 15.6\% gap is larger because the Krawczyk operator requires additional margin near the bifurcation.

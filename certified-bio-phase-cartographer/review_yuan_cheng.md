# Review: Certified Phase Atlases for Biological ODE Models

**Reviewer:** Yuan Cheng (Probabilistic Modeling & Stochastic Processes)
**Score: 6/10**

---

The Certified Phase Cartographer is fundamentally a deterministic validated-numerics system, and its engagement with probabilistic methods is limited to the optional GP-based adaptive exploration layer—which the problem statement explicitly characterizes as "engineering infrastructure" that decides where to look, never what to certify. This is a defensible architectural choice that keeps the core certification pipeline free of stochastic contamination, but it means the system has no formal framework for incorporating measurement uncertainty in a probabilistically rigorous way. When a systems biologist reports parameter values with uncertainty ranges, those ranges typically reflect Bayesian posterior credible intervals from MCMC fitting, not worst-case deterministic bounds. The system treats parameter boxes as deterministic set-membership constraints, which is conservative but potentially wasteful: a parameter region that has 99.9% posterior probability of being bistable would be treated the same as one with 50.1% probability, because the system can only certify or not certify—it has no mechanism for probabilistic qualification of regime claims.

The GP surrogate (Matern-5/2 ARD kernel, marginal likelihood optimization, boundary-focused acquisition function) for adaptive exploration is standard methodology well-suited to the task, but the implementation results reveal significant coverage gaps that a more sophisticated probabilistic approach might address. The Brusselator achieves only 84.4% coverage with 16 cells, and the Repressilator manages only 25% coverage with 3 cells—the uncertified gaps occur near Hopf bifurcation boundaries where regime identity is most uncertain. A probabilistic framework could assign graded confidence to regime claims near boundaries rather than leaving these regions as binary uncertified gaps. The optional extension E2 (moment-closure error bounds for stochastic kinetic models) is explicitly scoped as future work, which is honest but means the system cannot handle the stochastic models that are increasingly important in systems biology—particularly for low-copy-number gene expression where deterministic ODE models are known to be poor approximations. The Krawczyk contraction factors (0.661 for toggle switch, 0.802 for Brusselator, 0.833 for Sel'kov) reported in grounding.json are well within the contraction regime, but the convergence rate depends on the tightness of the initial interval enclosure, which is a deterministic choice that would benefit from probabilistic guidance about where equilibria are likely to be found. The evaluation plan's target of at least 95% coverage for p <= 4 parameters is honest but the current implementation achieves this only for 2-state models, and the gap to 3-state models (25% on the Repressilator) suggests the adaptive refinement strategy needs substantial improvement.
